Proteator

Proteator (http://proteator.appspot.com/) is a peptide mapping tool created at University of Freiburg.

It allows input of peptide data in form of text files (.csv, .tsv, .mztab, etc.). Upon input, data concerning sequence and topology of the proteins is retrieved from Uniprot, the peptides are mapped to their positions and the result is displayed in the web browser.

Proteator includes advanced analysis functionality such as visual display of fold changes or the ability to filter proteins for certain topological domains.

The finished visualization can be exported as .html file for ease of sharing.
